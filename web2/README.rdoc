= README

This project uses ruby 1.9.3 (see .rvmrc)

== Installation

=== Dependencies

* redis
  - on mac, `brew install redis`
  - on \*nix, `pacman -S redis` or similar

=== Gems

   $ bundle install

=== Database

   $ rake db:migrate

=== Resque (background job processing)

   $ COUNT=[NUMWORKERS] QUEUE=main rake resque:workers

== Running

   $ rails server
